# Vitallab
## Live Demo : https://vitallab.onrender.com/
Collaborative project by Durva Deorukhkar & Veronica Franco 

OVERVIEW:-

This project is a full-stack web application that allows users to browse, book, and manage health checkup tests online. It is designed to simplify the process of scheduling diagnostic tests and accessing reports digitally.

FEATURES:-

-User Features:
* User registration and login
* Browse health checkup test catalog
* Book tests online
* View and manage profile
* Access test reports
* Payment interface (basic implementation)
  
-Admin Features:
* Admin login page
* Admin dashboard UI (currently not fully functional)

TECH STACK:-
* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot
* Database: PostgreSQL
* Deployment:
  -Render (Frontend + Backend)
  -Railway (Database & Environment Variables)

LIMITATIONS:-
* Admin dashboard is not connected to the database
* Basic authentication (advanced security not implemented)
* Data privacy measures are minimal
* Payment system is not integrated with a real payment gateway

FUTURE SCOPE:-
* Connect admin panel with database
* Implement JWT authentication and role-based access control
* Integrate secure payment gateway (Razorpay/Stripe)
* Add appointment scheduling with time slots
* Enable email/SMS notifications
* Improve data security and encryption
* Add doctor/lab integration for report uploads
* Build mobile-friendly UI or app version
